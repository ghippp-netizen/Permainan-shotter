# Permainan-shotter
Permainan shooter ini adalah game tembak-menembak 2D bertema luar angkasa. Pemain mengendalikan pesawat roket yang dapat bergerak ke kiri dan ke kanan serta menembak musuh yang datang dari atas layar. Setiap musuh yang berhasil ditembak akan menambah skor pemain, sedangkan musuh yang lolos ke bawah layar atau menabrak pesawat akan mengurangi kesempatan bermain. Terdapat batas jumlah tembakan sehingga pemain harus menunggu beberapa saat untuk reload. Permainan berakhir dengan kemenangan jika skor mencapai target dan kekalahan jika batas kesalahan terlampaui.Selain musuh utama, terdapat asteroid yang menjadi rintangan tambahan dalam permainan. Asteroid bergerak dari atas ke bawah dan harus dihindari oleh pemain. Permainan dilengkapi dengan musik latar dan efek suara tembakan agar suasana lebih menarik. Kecepatan dan jumlah musuh membuat pemain harus fokus dan cepat bereaksi. Game ini melatih ketepatan, konsentrasi, dan strategi pemain dalam mengatur tembakan.
